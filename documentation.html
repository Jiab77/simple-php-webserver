Overview

This Simple PHP Web Server is an ultra-lightweight, single-file server designed to streamline development and offline work without having to spin up bloated modern local environments.

In less than 1,000 lines of highly optimized, clean PHP code, it packs premium capabilities, such as multi-threaded concurrency, clean automated routing fallback, advanced custom asset streaming, and built-in failure recovery.

CLI Command Flags

Run your server easily with full configuration control:

Flag Alternative Description
--port=[number] -p [number] Define target listening port. Defaults to 8000. Dynamic increments if occupied.
--host=[address] -h [address] Binding address interface. Defaults to 127.0.0.1.
--root=[path] -t [path] Document directory root mapping. Defaults to server's own folder.
--workers=[number] -w [number] Sets concurrency threads. Defaults to total detected CPU logical cores.
--debug -d Output profiling diagnostic and environment variables in the console logs.

Example usage:

php web/server.php --port=8080 --workers=4 --debug

Concurrency Capabilities

PHP standard built-in server is normally single-threaded, meaning assets (JS, CSS, images) block each other when loaded simultaneously.

This engine leverages PHP_CLI_SERVER_WORKERS (supported natively on PHP 7.4+) to orchestrate worker pools running on independent native threads corresponding to your CPU count. This guarantees instantaneous, lag-free loading, even for heavy assets!

Intelligent Asset Streaming

Serving text files is easy, but streaming binary content smoothly requires precise headers. The engine houses built-in MIME detection supporting over 30 file formats (from standard SVGs, WebPs, stylesheets up to TTFs/WOFF2 font structures and media clips).

In addition, it employs chunked binary piping read limits to prevent RAM exhaustion when streaming larger files.

Clean Routing Fallbacks

Standard built-in servers break when users hit non-existent folders or try custom URLs. This engine implements safe router fallbacks:

Translates dynamic routes safely into root base matching.
Generates a beautiful live dashboard if root base index files are missing.
Graccfully drops to the premium custom-styled 404 page rather than showing raw system errors.
